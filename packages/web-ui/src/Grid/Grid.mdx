### gutter
We don't provide any gutters, you have to implement it with negative margin in flex container and positive padding in flex items

<Playground>
  {() => {
    const Container = styled(Grid)`
      margin: -10px;
    `;
    const Item = styled(Grid)`
      padding: 10px;
    `;
    const Content = styled.div`
      border: 1px solid #cecece;
    `;
    return (
      <Container flex>
        <Item span><Content>flex item with 20px gutter</Content></Item>
        <Item span><Content>flex item with 20px gutter</Content></Item>
      </Container>
    );
  }}
</Playground>

### direction
Notice that grid works for "column" only when container's height is greater than flex items, such that there are space for distribution.

<Playground>
  {() => {
    const Container = styled(Grid)`
      height: 200px;
    `;
    const Item = styled(Grid)`
      border: 1px solid #cecece;
    `;
    return (
      <Container flex direction="column">
        <Item span={4}>span=4</Item>
        <Item span>span=true</Item>
        <Item span="auto">span=auto</Item>
      </Container>
    );
  }}
</Playground>
